Contributions are very welcome! Thanks.

If you've found a bug, please [open an issue](https://github.com/samcunliffe/rfmtbcmp/issues/new/choose) and then feel free to just fork and open a pull request if you know how to fix it.

If you want something that this tool doesn't do, check to see if there's already a [feature request](https://github.com/samcunliffe/rfmtbcmp/issues?q=state%3Aopen%20label%3A%22feature%20request%22) or a [closed feature request](https://github.com/samcunliffe/rfmtbcmp/issues?q=state%3Aclosed%20label%3A%22feature%20request%22). If not, please open an issue and ask.

I'm trying to keep the code as clean and idiomatically Go as possible.
I'm trying to keep test coverage as high as I can.
